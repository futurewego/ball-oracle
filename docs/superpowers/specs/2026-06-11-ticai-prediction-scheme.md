## 0. v2 优化要点（本次迭代，优先级高于下文冲突处）

v1 有几处"听着对、实则会翻车"，本轮修掉 6 个最高杠杆问题：

1. **国际足球评分尺度（修建模硬伤）** — v1 用"联赛基准μ"是错的：世界杯无联赛、跨洲球队几乎不交手。改为在**国际比赛结果数据集**（1872–2026，含友谊赛/预选赛/洲际杯/世界杯）上联合估计**全球统一**的 attack/defense 评分（带主场/中立场项），μ 重定义为"国际比赛全局基准"。跨洲样本稀疏 → 用**洲际(confederation)层级先验 + 强收缩**，避免弱样本队失真。

2. **校准悖论破解** — v1 要求"回测达标才上线 EV"，但 2026 世界杯**无法用自身赛前数据校准**，2022WC 迁移性又存疑 → 等校准完赛事都结束了。改：用 2022WC+2024Euro 训练**信息先验**并对 2026 队伍强收缩；把**整届世界杯当作在线校准期**，每个比赛日做贝叶斯更新评分；前台**始终展示带宽较大的概率区间**（而非点估计），诚实表达不确定性。EV 标记仍需达标，但概率展示不被校准卡死。

3. **价值定位诚实化（已实测，见 §8-P0）** — 竞彩主流盘返还率 ~86.3%（水位 14–16%，约书商 2–3 倍）。它**不是高效市场，而是公众资金驱动的"软盘"**：线不够锐、有系统性偏差（热门队/球星队被高估的 favorite-longshot bias），但**要先跨过 ~14–16% 的厚水位**。现实 edge 来自：① 公众偏差（冷门/平局常被低估）；② 冷门玩法（比分/半全场，水位分布更不均、更易错价）；③ 开盘早期 SP。**结论：正 EV 在主流盘稀有但非不可能；方案不承诺"稳定赢主流盘"，价值在避坑 + 抓偏差 + 冷门玩法。**

4. **产品价值重构** — 从"帮你找稳赢的注" → "**帮你看懂概率 + 避开烂注（识别负 EV 陷阱）+ 玩法教育 + 娱乐 PK**"。更诚实、天然合规、且降低用户亏损后的反噬。这是产品的真正卖点。

5. **数据源具体化** — SP 值**无官方开放 API**。方案：抓取 sporttery.cn 竞彩赔率页 / 聚合站(500.com、zgzcw)，赛前定时刷新 + 缓存 + 时间戳；历史比分与国际赛果用公开数据集冷启动。**"数据可得性与稳定性"列为上线前 P0 验证项**（比模型更可能卡住项目）。

6. **MVP 再收紧 + 成本** — 第一波就 胜平负 + 总进球 概率展示，明确页面/cron/成本（见 §7）：LLM 文案每场 1 次调用，月成本近似可忽略；真正成本是**数据抓取稳定性 + 回测工程工时**。

## 1. 方案总览

用**一个进球数联合分布模型**（Dixon-Coles 双泊松）拟合每场比赛的 P(主进 i, 客进 j) 矩阵，由该矩阵**解析导出全部 5 大玩法**的概率（胜平负/让球/比分/总进球/半全场均是同一矩阵的不同求和方式）。再用去水位后的市场隐含概率求 EV，定位被低估选项。**关键纪律：模型概率必须先过可靠性校准，前台只展示"概率区间+回测样本量"，不打包推单、不诱导投注。**

**产品定位（v2 校准）**：竞彩 SP 是相对有效的市场，本品**不承诺帮你稳赢主流盘**。真正价值=「**看懂概率 + 避开烂注（识别负 EV 陷阱）+ 玩法教育 + 跟 AI 比准的娱乐 PK**」。现实可挖的 edge 集中在冷门玩法(比分/半全场)与开盘早期 SP，而非主流胜平负。这个定位既诚实、又天然合规。

## 2. 核心预测模型

**2.1 Dixon-Coles 双泊松**
λ_h = α_h·β_a·γ·μ，λ_a = α_a·β_h·μ（α进攻、β防守、γ主场/中立场项、μ=**国际比赛全局基准**，见 §0.1）。世界杯多为中立场，γ→1 或按"东道主/地理临近"微调。对低比分 (0,0)(1,0)(0,1)(1,1) 乘 τ 修正因子，由 MLE 与 ρ 参数共同拟合，消除独立泊松对低比分的低估。截断 N=10 归一。

**2.2 全场→半全场（修正①：禁止独立笛卡尔积）**
不把"半场边际"与"全场边际"相乘。正确做法：将比赛切成上下两段，(X_HT,Y_HT)~Pois(λ·c) 与 (X_2H,Y_2H)~Pois(λ·(1−c))，**两段独立泊松相加得全场**；半全场 9 宫格从 (HT, HT+2H) 的**联合分布**导出，c≈0.45 由历史半场数据校准。如此 HT 与 FT 强相关被保留，不再系统性高估逆转/崩盘。

**2.3 让球三分重定义（修正②）**
主让 n 球：以 (i−n) 对 j 重判。让胜=Σ_{i−n>j}P(i,j)，**让平=Σ_{i−n=j}P(i,j)**，让负=Σ_{i−n<j}P(i,j)。让 1 时须 i−1=j 才计让平，平局类别整体重定义。

**2.4 参数估计（小样本 / 国际足球，v2 强化）**
训练集=**国际比赛结果数据集**（1872–2026，含友谊赛/预选/洲际杯/世界杯，公开数据集可得）。分层加权 MLE：时间衰减 w_t=e^{−α(t_now−t)}(α≈0.003/天) × 赛事权重 w_c（正赛1.0/预选0.6/洲际0.5/热身0.2）。冷启动用 FIFA/World Football ELO **中心化**映射（修正③）：α_i = exp((ELO_i−ELO_mean)/400·ln10)，进攻强度为相对量纲（全局均值≈1），再 MLE 微调。弱样本队（小国、跨洲少交手）加 L2 正则**拉向其所属洲际(confederation)均值**（层级先验），而非单一全局均值——既防过拟合又保留洲际实力差。世界杯期间每个比赛日**贝叶斯更新**评分（在线校准，见 §0.2）。

**2.5 LLM 受控调 λ**
LLM 只乘 λ、不改结构。接口：factor_h/factor_a∈[0.85,1.15]，reason≥20字结构化理由，confidence∈[0,1]，低置信自动降权 factor_eff=1+(factor−1)·confidence；越界截断并记日志供回测审计。

**2.6 校准与回测**
回测集：2022世界杯+2024欧洲杯滚动回测（禁未来信息）。指标 Brier(胜平负)<0.22、LogLoss<0.95。**输出概率必须过 Isotonic/Platt 后校准**，可靠性曲线偏离>5% 重新校准。未过校准的概率严禁对外（修正⑩）。

## 3. 五大玩法预测与价值策略

- **胜平负/让球**：模型三向概率 vs 1/SP 找低估。强主场对弱队、主让1球 SP>1.6 常正 EV。**平局信号仅在"实力相近+回测验证"下提示，删除"默契保平/打假球"叙事（修正⑥）**——该类纯故事不上线，最多作弱信号待回测。
- **比分**：选 SP 高估而非最可能比分。EV=p×SP−1，关注 1:0、2:1、0:0 及"胜/平其它"高 SP 尾部，小额配置。
- **总进球**：由总 λ 导出 8 档分布直方图，标 EV>0 档位。世界杯淘汰赛均值低（2.3–2.5），慎押高档。
- **半全场**：**先修 2.2 联合分布再谈（修正⑦）**。默认不前推，仅当联合分布识别明确节奏特征才提示，标注方差极大。删除"SP25-60 值得押"类高赔诱导。

## 4. 从概率到投注

**4.1 去水位**：主力 Shin 法 p_i=(√(z²+4(1−z)q_i)−z)/(2(1−z))，q_i=(1/SP_i)/Σ(1/SP)，对大冷门去水更准；Multiplicative 作校验。RTP=1/Σ(1/SP)。z 由历史 SP 反推（约0.01–0.03），**建库实测校准**。

**4.2 EV 与价值判定**：EV=p_model×SP−1。标价值需同时满足 EV>8%（初期保守）、该场景回测≥30场且 Brier 达标、|p_model−p_market|/p_market<40%。分歧>40% 触发人工复核——**优先怀疑模型**（市场是千万资金聚合信号）。

**4.3 分数凯利（修正④）**：f*=EV/(SP−1)，取 1/4 Kelly。**2% 是组合层总暴露约束，非逐注硬顶**：多注并行按组合 Kelly 控制总仓位 ≤bankroll×2%，避免高 EV 场被无谓砍仓、也避免并行暴露失控。

**4.4 串关与封顶（修正⑤）**：命中率=Π(p_i) 指数衰减，凸性损失+方差惩罚下各场 EV 均正时**单关之和优于串关**。封顶=｛2-3关20万/4-5关50万/6+关100万｝。真实 EV_real=Π(p_i)×min(2·Π SP_i, cap)/2−1。**有效示例**：5 关各 SP=15，理论 2×15⁵≈152万 > 50万封顶，削顶后 EV 大幅缩水——高赔串关必跑封顶检查。

## 5. AI 推荐与用户呈现

**每场页（三层）**：① 一眼结论（模型倾向方向+信心星级+一句中文锐评，由 Claude 把数字转文案）；② 价值看板（玩法/选项/**概率区间+样本量**/SP/价值标记）——**EV 与"高价值"仅在过校准后展示，否则只显概率区间，不显伪精度 EV（修正⑩）**；③ 折叠：31比分热力图、半全场9宫格、3条驱动因素。
**AI 信息卡（非推单）**：每日输出"高 EV 信息位"列表，措辞为"模型倾向/参考概率"，**删除"精选N注/2串1推荐/晒单分享钩子"（修正⑧）**——改为纯信息展示+用户自选，不打包组合。
**用户 PK**：纯预测、不需真实投注，AI 同步提交，结算命中率与模拟收益，扩池且规避合规风险。

## 6. 合规与风控红线

**必须不做**：代购/资金归集/代下注、承诺收益保本、"稳赚/必中/跟单"话术、向未成年提供、跳转非官方购彩、**打包推单售卖（修正⑧统一为信息展示+用户自选）**。
**必须做**：展示公开数据/概率区间/SP对比；置信分级（低置信不前推）；数据时间戳（>30分钟橙色警告）；注明"未含最新阵容变动"。
**三处免责（缺一不可）**：① 启动页置顶"仅供数据分析参考，不构成投注建议，理性投注，未满18禁用"；② 每条结果下"模型概率≠中奖概率，竞彩长期返奖率<100%，亏损正常"；③ 注册强制勾选"我了解本品为信息工具，不代购、不承诺收益，自负投注责任"。
**数据合规**：优先官方体彩接口；第三方抓取评估 robots/版权；用户数据 PIPL 合规、禁售第三方。

## 7. MVP 分波落地（修正⑨：3-4周不现实，砍范围）

**架构对接**：
- **Next.js（App Router）**：比赛列表/详情（SSR+ISR）、信息卡页（SSG）、PK（客户端）。
- **Supabase**：matches（SP、概率、校准后标记）、predictions（用户预测）、daily_info（每日信息位）。
- **预测引擎 API（Python/FastAPI）**：赛前 6 小时跑 Dixon-Coles，输出 5 玩法概率+（校准后）EV 写 Supabase。
- **Claude**：只做文案（数字→中文锐评），固定 prompt 模板，每场一次调用，成本可控。

**数据刷新节奏（cron）**：赛程每日 1 拉；SP 值赛前 T-24h / T-6h / T-1h 三档刷新 + 缓存 + 时间戳；模型 T-6h 跑一次、阵容大变可手动重跑；赛后拉真实比分结算 PK 与评分贝叶斯更新。
**成本量级（第一波）**：LLM 文案——单场赛日约 30-50 场 × 每场 1 次 Sonnet 调用（输入数据+模板约几千 token），日成本约 ¥1-3 量级，**整届可忽略**；主成本集中在 ① 数据抓取稳定性运维 ② 回测/校准的一次性工程工时。Vercel + Supabase 免费/低档起步即可。

## 8. 关键待验证项（v2 重排优先级 + P0 实测结论）

- **【P0·数据可得性 — ✅ 已验证可行】** SP 值**无官方开放 API**，但数据可得，两条路：① **抓取** sporttery.cn 竞彩赔率页 / zgzcw / 500.com（免费但页面 JS 重、需维护、有版权/反爬风险）；② **付费第三方聚合 API**（聚合数据、飞鲸 feijing88、即时数据等，含竞彩 SP 实时数据，稳定性更好）。**建议 MVP 用付费聚合 API 保稳定，抓取做兜底。** 竞彩可投注时段：周一至周五 11:00–22:00、周六日 11:00–23:00（数据刷新窗对齐此区间）。赛果/国际历史赛果用公开数据集（International football results 1872–2026 类）冷启动。
- **【P0·水位实测 — ✅ 已得关键结论】** 竞彩主流盘(胜平负)**返还率约 86.3%**（近年从 88–89% 降至此），即 **Σ(1/SP) ≈ 1.16、水位约 14–16%**。对比：顶级书商 ~2–3%、软盘书商 ~6–8%。**含义（改写价值判断）**：竞彩不是"高效市场让你赢不了"，而是**公众资金驱动的"软盘"——线本身不够锐、存在系统性偏差（热门队/球星队被高估的 favorite-longshot bias），但要先跨过 ~14–16% 的厚水位**。所以正 EV 在主流盘**稀有但非不可能**，真正机会=「公众偏差 + 冷门玩法(比分/半全场水位分布更不均) + 开盘早期 SP」。⚠️ 各玩法/各场水位会浮动，建库时按场抽样确认。
- **【P1·回测达标 — ✅ 已初测】** 实测(2026-06-11)：多分类 Brier=Σ_k(p_k−y_k)²(范围0~2)，均匀瞎猜=0.667。本模型 **Euro2024 Brier=0.544/LogLoss=0.919、WC2022 Brier=0.601/LogLoss=1.040**，两届均胜瞎猜、Euro 接近市场级 → **模型有真实信号、决策门通过**。⚠️ 之前误写的"Brier<0.22"是混淆度量约定，三分类求和Brier下不可达，已废止。**真正对标是竞彩 SP 隐含概率的 Brier（需 SP 数据）——跑赢瞎猜≠跑赢市场**。要稳定正 EV 需净 edge > ~14% 水位，故主流盘门槛极高，EV 标记初期从严(>8%)。
- **【P1·迁移性】** 验证 2022/2024 训练的先验对 2026 阵容/新生代球队的迁移性；不足则加大收缩、加宽前台概率区间。

## 9. 完整性审查补漏（v2.1，按"谁会先炸"排序）

v1/v2 在建模与合规上做扎实了，但有 9 个**整段缺失**的盲区，补齐如下。

### 商业与生存（最致命，Marvin 视角）

**9.1 盈利模式（v1/v2 完全没提）。** 合规已禁止代购/推单/收益分成 → 那靠什么活？可行且合规的路径：① **Freemium 订阅**——免费看基础概率+PK，付费解锁深度（全玩法 EV、比分热力图、AI 详细推理、历史 track record、自定义赛前提醒）；整届世界杯小额买断或月订阅。② 后期 **B2B 数据服务**——把校准后的概率作为数据卖给内容方/媒体。③ **绝不碰**：代购返佣（违法）、付费推单/保盈会员（违规）。MVP 先免费起量，订阅在第二波。

**9.2 竞品与差异化（没提，但这是"凭什么用你"）。** 市面已有唯彩看球、球探网、捷报、雷达体育、球天下等做赔率分析/AI 预测。多数走**黑盒推单/红人晒单**模式（合规灰、信任低、夸大胜率）。ball-oracle 的差异化打**信任牌+工具牌**：透明概率 + 校准可验证 + **诚实不承诺赢** + 一套模型统一 5 玩法 + LLM 人话解释 + 跟 AI 公开 PK。"诚实"本身就是差异化——在一堆"稳赚跟单号"里做"帮你看懂+少亏"的工具。

**9.3 获客/传播（推单被禁后怎么传）。** 不能晒单 → 靠：免费概率工具口碑、PK 排行的社交裂变、**AI 公开战绩页**（诚实 track record 本身是内容）、世界杯话题的赛前预测/赛后复盘图文（发小红书/公众号，是"内容"不是"推单"，注意边界）。

### 建模与数据（会让预测系统性偏）

**9.4 WC2026 物理因素（美加墨特有，直接影响进球，没建模）。** ① **旅行疲劳**：横跨多时区、长途飞行，连续客场/长途队 λ 下调。② **海拔**：墨西哥城/托卢卡/瓜达拉哈拉高海拔，不适应队体能与传跑受影响。③ **高温**：6-7 月美南/墨西哥酷热 + 为欧洲转播的正午开球 → 节奏慢、进球可能降低、有补水暂停。④ **东道主主场**：美、加、墨三国主场 γ>1（真主场+裁判+观众），其余中立场 γ→1。这些一部分进模型特征（场馆海拔/气温/旅行里程），一部分走 LLM 受控调 λ。

**9.5 串关跨场相关性 + 死亡之组默契球（串关 EV 会算错）。** Dixon-Coles 是单场模型，串关默认跨场独立。但**小组末轮**两队"打平即携手出线"会产生默契/正相关，使联合命中率 ≠ 概率连乘（历史："希洪之耻"）。第三轮同组同时开球削弱但未消除动机偏差。方案：对"同组末轮+出线形势敏感"场次，串关计算加相关性折扣/警示，或直接**排除此类场入串**；LLM 层标注"默契风险/死亡之组"。

**9.6 阵容/伤停数据源（LLM 调 λ 的输入依赖，没指定来源）。** §2.5 让 LLM 按伤停调 λ，但赛前预测阵容/伤停从哪来？需专门 feed（聚合 API 是否含、或媒体/球队官方）。**若无可靠源，LLM 调整退化为基于公开新闻的弱信号，必须降权**，别让幻觉阵容污染概率。

### 统计严谨（决定"能不能说模型有效"）

**9.7 CLV（收盘线价值）作为 edge 金标准（没提）。** 一届世界杯仅 ~104 场，**单届 ROI/盈亏方差极大、统计上证明不了模型好坏**。专业做法是看 **CLV**：若被标记选项在收盘时 SP 下降（市场向你靠拢），说明你领先于线 = 真 edge，比战绩可靠得多。方案加 CLV 跟踪，并**对内对外都不许用单届战绩下"模型很准"的结论**。

**9.8 多重比较假阳性（多玩法×多场→必然刷出假价值）。** 同时扫 5 玩法×几十场×多选项，纯随机也会冒出一堆"高 EV"。需阈值收紧/FDR 控制/最小样本门槛，否则价值标记一堆是噪声。已在 §4.2 留口，此处显式列为纪律。

### 合规深化

**9.9 责任博彩功能（超出免责文案的实际功能）。** 比免责声明更进一步、也更差异化：可选的"我已投注额"自记 + 连亏提示 + 冷静期/自我屏蔽、强未成年限制、**绝不展示"晒收益"组件**。这既是合规护城河，也强化 9.2 的"负责任"差异化。

### 对 MVP 的影响（增量）

第一波**新增必做**：东道主主场/海拔/高温作为模型特征（数据现成、低成本）；呈现层落实 9.7（不晒单届战绩、显概率区间）。
第一波**可后置**：订阅(9.1)、串关相关性(9.5)、CLV 跟踪(9.7) 随对应玩法上线再做；阵容源(9.6) 若 P0 阶段确认无稳定源，则 LLM 调 λ 在第一波直接关闭、只用纯数据概率。

## 附录 A：已核实的竞彩足球规则基线（2026世界杯适用）

5大玩法（均以**全场90分钟+伤停补时**判定，不含加时/点球）：
1. **胜平负(SPF)**：主队 胜/平/负，3选项。
2. **让球胜平负(RQSPF)**：整数让球（如主-1、主+2）后再判胜平负，3选项。让球数赛前公布。
3. **比分**：31选项。主胜12个具体比分 + 胜其它；平4个(0:0,1:1,2:2,3:3) + 平其它；负12个对称 + 负其它。
4. **总进球数**：8选项 0,1,2,3,4,5,6,7+。
5. **半全场胜平负**：9组合（上半场×全场 的 胜/平/负）。

赔率与投注：
- 每选项有公布 SP值，单注奖金=2元×SP值连乘。
- 单关（单场）=浮动奖金；串关（≥2场）=固定奖金，SP锁定于购买时刻。单关仅限标"单"场次。
- 过关方式：胜平负&总进球 支持 2×1~6×1；比分&半全场 仅 2×1、3×1；自由过关支持任选 M×1(2~8)。
- 奖金封顶：2-3关20万 / 4-5关50万 / 6关及以上100万元/单注。
- SP值含水位（Σ(1/SP)>1），返奖率约65-70%档（精确值待实测校准）。
